# Supertonic ONNX Models

Place the Supertonic 3 ONNX models and voice styles here.
You need the following structure:

```
assets/
  onnx/
    duration_predictor.onnx
    text_encoder.onnx
    vector_estimator.onnx
    vocoder.onnx
    tts.json
    unicode_indexer.json
  voice_styles/
    M1.json
    ... (other styles)
```

You can download the models from: https://huggingface.co/Supertone/supertonic-3
